アコーディオン

ある程度のデザインを加えたアコーディオンです。そこに、メニューの内容も加えてみました。

まずはメニューを単純に列挙

{{ if .Site.Menus.main }}
    {{ range .Site.Menus.main.ByWeight }} {{ if .HasChildren }}
  • {{ .Name }}
      {{ range .Children }}
    • {{ .Name }}
      • {{ end }}
    • {{ else }}
  • {{- .Name -}}
    • {{ end }} {{ end }}
{{ end }}

次に、具体的に実装してみたメニュー

{{ if .Site.Menus.main }}
    {{ range .Site.Menus.main.ByWeight }} {{ if .HasChildren }}

  • {{ .Name }}

      {{ range .Children }}
    • {{ .Name }}
      • {{ end }}
    • {{ else }}

  • {{- .Name -}}

    • {{ end }} {{ end }}
{{ end }}
